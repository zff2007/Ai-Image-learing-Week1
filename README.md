# AI-Image-learning-week1
图片AI开发助理实习 | 第1周代码库

##  项目介绍
本项目是 **图片AI开发助理实习第1周** 的完整学习成果汇总，从 Python 基础语法入门，逐步掌握 PIL 图片处理、OS 文件系统操作，最终整合实现 **数据集图片预处理工具**，并完成 Git 版本控制与 GitHub 云端托管，形成可复用、可展示的编程作品集。

##  技术栈
- **编程语言**：Python 3
- **图片处理库**：Pillow（PIL）
- **文件系统库**：Python 内置 os 库
- **版本控制**：Git + GitHub

##  项目结构
AI-Image-learning-week1/
├─ Day1_Python 基础语法 / # Python 基础：变量 / 循环 / 判断，偶数统计脚本
├─ Day2_PIL 基础操作 / # PIL 入门：图片打开、显示、保存，格式转换与图像混合
├─ Day3_PIL 核心操作 / # PIL 高级：缩放、裁剪、旋转、通道分离 / 合并，透明图处理
├─ Day4_os 文件路径操作 / # OS 库：文件夹遍历、路径拼接、批量文件筛选
├─ Day5_数据集预处理工具 / # 综合项目：批量图片预处理（原版，整合 Day2~Day4 知识）
└─ Day6_查漏补缺_工具优化 / # 项目优化版：bug 修复、异常处理细化、代码注释完善 + Git 提交

##  核心功能（数据集图片预处理工具）
- **批量重命名**：自动将图片命名为 `image_001.jpg` 格式，方便排序与管理
- **尺寸标准化**：统一将图片缩放至 500×500 像素，保持比例不变形
- **格式统一**：自动将 PNG/BMP 等格式转换为 JPG 格式
- **透明图兼容**：自动处理 RGBA/P 模式透明图，转为白底 JPG 避免黑边
- **健壮性保障**：异常捕获机制，单张图片处理失败不中断整体流程
- **安全处理**：原始图片不修改，处理后文件自动存入新目录

##  快速运行
1. 安装依赖：
```bash
pip install pillow
```
2. 打开预处理工具脚本（Day5_数据集预处理工具 / 或 Day6_查漏补缺_工具优化 / 下的 .py 文件）
3. 修改代码中 SOURCE_DIR 为你的本地图片文件夹路径
4. 运行脚本，自动完成批量处理
